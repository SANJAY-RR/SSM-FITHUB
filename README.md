# SSM FitHub - Gym Management System

## 📌 Project Overview

SSM FitHub is a web-based Gym Management System developed to simplify gym member management and performance tracking.  
This project helps gym administrators manage members, trainers, memberships, attendance, workout plans, and overall gym activities efficiently.

The system is designed with a user-friendly interface and secure backend management functionalities.

---

## 🚀 Features

### 👤 User Management
- Member Registration
- Secure Login System
- Profile Management
- Password Recovery

### 🏋️ Gym Management
- Add / Update Members
- Trainer Management
- Membership Plans
- Workout Tracking
- Attendance Monitoring

### 📊 Dashboard & Reports
- Admin Dashboard
- Member Statistics
- Performance Monitoring
- Activity Tracking

### 🔒 Security Features
- Authentication System
- Session Handling
- Secure Login Validation

## ⚙️ Installation Steps

### 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/SSM-FITHUB-FINAL-YEAR-MINI-PROJECT.git
```

---

### 2️⃣ Move Project to XAMPP htdocs

```bash
C:\xampp\htdocs\
```

---

### 3️⃣ Import Database

- Open phpMyAdmin
- Create database
- Import SQL file

---

### 4️⃣ Start Server

Start:
- Apache
- MySQL

using XAMPP Control Panel.

---

### 5️⃣ Run Project

Open browser and visit:

```bash
http://localhost/SSM-FITHUB/
```
